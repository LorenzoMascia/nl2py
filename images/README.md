# Images Directory

This directory contains images used in the project documentation and README.

## Current Files

- **logo.svg** - NL2Py logo used in documentation website and README

## Adding GUI Screenshot

To add a screenshot of the GUI to the README:

1. Launch the GUI: `nl2py-gui`
2. Take a screenshot of the interface
3. Save it as `gui-screenshot.png` in this directory
4. Uncomment the screenshot line in `README.md`:
   ```markdown
   ![NL2Py GUI](images/gui-screenshot.png)
   ```

## Logo Usage

The logo is available in:
- `images/logo.svg` - For README and general use
- `docs/assets/images/logo.svg` - For documentation website

Both files are identical and can be updated if you want to use a custom logo.
