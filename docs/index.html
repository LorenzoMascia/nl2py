What is NL2Py?

NL2Py is a natural language to Python compiler that translates human-readable commands into executable code for cloud services, databases, APIs, and infrastructure management.

NLP-Powered Matching

Uses TF-IDF similarity to intelligently match your commands to the right methods, even with varied phrasing.

35+ Service Modules

Pre-built integrations for AWS, GCP, Azure, Kubernetes, Docker, databases, messaging systems, and more.

Parameter Extraction

Automatically extracts parameters from your natural language commands using pattern matching.

Web GUI & CLI

Choose between a beautiful Gradio web interface or command-line tools for your workflow.

Extensible

Easy-to-follow module interface. Add your own service integrations in minutes.

Production Ready

Docker support, configuration management, and security best practices built-in.

See It In Action

Write natural language commands and get Python code instantly.

# Natural Language Input:
"create compute instance web-server in zone us-central1-a"

# Generated Python Code:
compute_instance_create(name='web-server', zone='us-central1-a')

# Another example:
"upload file report.pdf to s3 bucket my-backup"

# Generated:
s3_upload_file(file_path='report.pdf', bucket='my-backup')
# Natural Language Input:
"connect to postgres database myapp"

# Generated Python Code:
connect(database='myapp')

# Another example:
"set redis key user:123 to John Doe"

# Generated:
set_key(key='user:123', value='John Doe')
# Natural Language Input:
"send message to slack channel general with text Deploy completed"

# Generated Python Code:
send_message(channel='general', text='Deploy completed')

# Another example:
"publish message to kafka topic orders"

# Generated:

# Natural Language Input:
"list all docker containers"

# Generated Python Code:

# Another example:
"deploy kubernetes deployment nginx with 3 replicas"

# Generated:

Quick Start

  1. Install NL2Py 
    pip install nl2py
pip install gradio  # For the web GUI
  2. Launch the GUI 
    nl2py-gui

    Opens at http://localhost:7860

  3. Write natural language commands 
    create s3 bucket my-data-bucket
list all docker containers
send slack message to channel devops
  4. Get executable Python code! 
    s3_create_bucket(name='my-data-bucket')
container_list()
send_message(channel='devops')

How It Works

NL2Py uses a multi-stage pipeline to transform natural language into code.
