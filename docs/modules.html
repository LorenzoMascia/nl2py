Modules Reference

NL2Py includes 35+ pre-built modules for interacting with cloud services, databases, messaging systems, and more.

Module Dependencies: Each module may require additional Python packages. Install only what you need to keep your environment lean.

AWS Module

Comprehensive AWS services integration including EC2, S3, Lambda, DynamoDB, and more.

Installation

pip install boto3>=1.26.0

Configuration

# nl2py.conf
[aws]
region = us-east-1
access_key_id = AKIA...
secret_access_key = ...

# Or environment variables
export AWS_ACCESS_KEY_ID=AKIA...
export AWS_SECRET_ACCESS_KEY=...
export AWS_DEFAULT_REGION=us-east-1

Example Commands

Natural Language Generated Code
create ec2 instance web-server ec2_create_instance(name='web-server')
list all ec2 instances ec2_list_instances()
create lambda function my-func lambda_create_function(name='my-func')
put item in dynamodb table users dynamodb_put_item(table='users')

Google Cloud Module

Integration with GCP services including Compute Engine, Cloud Storage, BigQuery, Pub/Sub, and more.

Installation

pip install google-cloud-compute>=1.14.0
pip install google-cloud-storage>=2.10.0
pip install google-cloud-bigquery>=3.11.0
pip install google-cloud-pubsub>=2.18.0

Configuration

# Using service account
export GOOGLE_APPLICATION_CREDENTIALS=/path/to/service-account.json
export GCP_PROJECT_ID=my-project
export GCP_DEFAULT_ZONE=us-central1-a

Example Commands

Natural Language Generated Code
create compute instance my-vm in zone us-central1-a compute_instance_create(name='my-vm', zone='us-central1-a')
list compute instances compute_instance_list()
create storage bucket my-data storage_bucket_create(name='my-data')
publish message to pubsub topic events pubsub_publish(topic='events')
